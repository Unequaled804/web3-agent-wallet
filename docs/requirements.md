# 需求描述

设计实现一个 AI Agent 使用的加密货币钱包

## 背景

加密货币钱包指可以管理加密资产、进行区块链交易的钱包应用。本题目可以具体为某个特定区块链（如以太坊测试链）的钱包。
AI Agent 指有一定智能及自主决策能力的智能体，如 OpenClaw, Claude Code, Codex, Hermes, Cursor 等。
现在希望为 AI 智能体场景设计实现一个钱包 Demo。

## 要求

- 思考 AI Agent 钱包的用户画像与使用场景，输出文档。
- 思考设计 AI Agent 钱包时需要重点解决哪些问题（不超过3个），输出文档。
- 完成 AI Agent 钱包的架构设计，输出文档
- 完成 AI Agent 钱包的代码实现，输出代码，要求代码可以实际运行。
